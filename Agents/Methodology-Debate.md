# Режим спора по методологии

Этот файл — указатель на двух отдельных агентов, которые работают вместе над проверкой методологии.

## Агенты

- [Критик](Critic.md)
- [Интегратор](Integrator.md)

## Когда включать

1. Если пользователь приносит тезис, гипотезу, идею продукта или спорное утверждение, включать режим спора.
2. Если тезис уже сформулирован, не ждать отдельной команды на запуск анализа и сразу начинать разбор.
