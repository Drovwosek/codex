# Агенты

Этот файл хранит рабочие правила и предпочтения.

## Правила

### Нейминг

1. Имена папок должны быть на английском языке и начинаться с заглавной буквы; существующие служебные файлы рабочего пространства могут сохранять свои привычные имена, если это нужно инструментам.

### Язык

2. Все скиллы и инструкции должны быть написаны на русском языке.

### Режим спора

3. Режим спора по методологии хранится в отдельном файле: [Methodology-Debate-Mode.md](Agents/Methodology-Debate-Mode.md).
4. Отдельные агенты режима спора хранятся в файлах: [Critic.md](Agents/Critic.md) и [Integrator.md](Agents/Integrator.md).

### Хранение правил

5. Не создавать отдельный файл для каждого правила без необходимости.
6. Если правило новое, записывать его в этот файл.
7. Если правила можно логично сгруппировать, объединять их в один раздел.
8. Если новое правило относится к уже существующей группе правил, добавлять его в эту группу.
9. Не складывать тематические документы в `Memory`, если для них уже есть отдельная корневая папка.
10. Использовать `Memory` для памяти, правил, контекста и вспомогательных указателей, а не для основного содержания тематических разделов.
