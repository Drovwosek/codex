# Skills

This folder is the single home for Codex skills and the materials they come from.

## Layout

- `local/` - locally adapted skills, notes, and workflow docs.
- `sources/` - upstream sources, plugin snapshots, and imported references.

## Goal

Keep everything related to skills under `skills/` so the workspace has one predictable place for:

- skill definitions
- adapted variants
- upstream source material
- documentation about how those skills are used

If you later move files in from other locations, preserve this split between `local` and `sources`.
