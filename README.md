# Рабочее пространство Codex

Репозиторий объединяет рабочий контекст, предметные знания, повторяемые процессы, роли анализа, проекты и внешние исходные материалы.

## Структура

- [Workspace](Workspace/README.md) - правила, текущий контекст и карта репозитория.
- [Knowledge](Knowledge/README.md) - общие предметные знания, не привязанные к отдельному проекту.
- [Skills](Skills/README.md) - повторяемые рабочие процессы Codex.
- [Agents](Agents/README.md) - роли и режимы анализа.
- [Projects](Projects/README.md) - материалы отдельных инициатив.
- [Sources](Sources/README.md) - внешние выгрузки и первичные материалы.

Все внутренние ссылки и инструкции должны использовать пути относительно корня репозитория.
