# Навыки

Раздел хранит повторяемые рабочие процессы Codex. Каждый навык находится в отдельной папке и имеет точку входа `SKILL.md`.

Навыки могут ссылаться на общие правила из `Workspace`, знания из `Knowledge` и роли из `Agents`, но не должны копировать их содержимое без необходимости.

## Связи

- `Workspace` задает общие ограничения и текущий контекст.
- `Knowledge` предоставляет определения и предметные модели.
- `Agents` предоставляет позиции для анализа и проверки решений.
- `Projects` хранит результат применения навыков к конкретной инициативе.

Таким образом, навык остается повторяемым процессом, а изменяемый контекст и знания подключаются к нему по ссылкам.
