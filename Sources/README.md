# Исходные материалы

Раздел предназначен для внешних выгрузок, снимков, импортированных документов и других первичных материалов.

Исходники не считаются долговременной памятью или готовым знанием. Если материал важен для работы, его выводы нужно оформить в `Knowledge`, `Workspace` или соответствующем проекте.
