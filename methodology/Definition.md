# Definition

Эта методология представляет собой гибрид двух источников:

1. Текущие продуктовые и бизнес-процессы, которые используются в компании.
2. AJTBD Вани Замесина.

Цель методологии - совместить текущую реальность компании со структурным подходом Jobs To Be Done так, чтобы продуктовые решения были:

- привязаны к тому, как компания уже работает;
- согласованы с потребностями клиентов и реальным спросом;
- достаточно явными, чтобы их можно было обсуждать, оспаривать и улучшать;
- разложены на маленькие проверяемые предположения.
